# GoldBar — 实时金价菜单栏应用

在 macOS 菜单栏中实时显示当前黄金价格（人民币/克）。

## 功能特性

- 🟡 **实时金价显示** — 菜单栏中显示最新黄金价格 (RMB/g)
- 🌐 **自动汇率转换** — 自动获取 USD/CNY 汇率，将美元/盎司转换为人民币/克
- ⚙️ **可配置设置** — 支持修改 API Key、切换汇率模式
- 💾 **低资源占用** — 后台静默运行，定时拉取数据
- 🔒 **菜单栏专属** — 不占用 Dock 空间，仅在菜单栏显示

## 系统要求

- macOS 13.0 (Ventura) 或更高版本
- Apple Silicon (M1/M2/M3/M4) 或 Intel Mac
- 互联网连接

## 安装

### 方式一：直接运行

```bash
cd GoldBar
./build.sh release
open build/GoldBar.app
```

### 方式二：拖入应用程序文件夹

编译完成后，将 `build/GoldBar.app` 拖入 `/Applications/` 文件夹。

### 开机自启（可选）

1. 打开「系统设置」→「通用」→「登录项与扩展」
2. 点击「+」添加 GoldBar.app

## 使用说明

### 菜单栏显示

启动后，你将在菜单栏右上角看到类似以下内容：

```
Au ¥895.2/g
```

- `Au` — 黄金化学符号
- `¥895.2/g` — 每克黄金的人民币价格

### 菜单功能

点击菜单栏图标，展开下拉菜单：

| 菜单项 | 功能 |
|--------|------|
| 黄金: $XXXX.XX/oz | 显示原始美元/盎司价格 |
| 汇率: X.XXXX USD/CNY | 当前使用的汇率 |
| 更新时间: HH:MM:SS | 最后一次成功获取数据的时间 |
| 立即刷新 (`⌘R`) | 手动触发数据刷新 |
| 设置... (`⌘,`) | 打开设置窗口 |
| 退出 GoldBar (`⌘Q`) | 退出应用 |

### 设置

打开设置窗口可以修改以下配置：

- **API Key** — AllTick 平台的 API 密钥。默认已预置一个免费密钥
- **汇率模式**
  - *自动获取（推荐）* — 每小时自动从免费汇率 API 获取最新 USD/CNY 汇率
  - *手动输入* — 手动输入固定汇率值
- **手动汇率** — 仅在手动模式下启用，输入你希望使用的 USD/CNY 汇率

修改后点击「保存」即刻生效。

## 数据来源

- **黄金价格**: [AllTick](https://alltick.co) 金融数据 API — 实时贵金属报价
- **美元汇率**: [Exchange Rate API](https://open.er-api.com) — 免费汇率数据

## 价格计算

```
人民币/克 = (美元/盎司 × USD/CNY汇率) ÷ 31.1034768
```

其中 `31.1034768` 为 1 金衡盎司对应的克数。

## 常见问题

### Q: 价格显示 `--.-/g` 是什么问题？
A: 表示暂时无法获取金价数据。请检查：
1. 网络连接是否正常
2. 在设置中确认 API Key 是否有效
3. 是否超过了 API 请求频率限制（免费版每分钟 10 次）

### Q: 汇率数据多久更新一次？
A: 自动模式下每小时更新一次。你也可以在设置中手动输入固定汇率。

### Q: 如何获取自己的 API Key？
A: 访问 [AllTick 官网](https://alltick.co) 注册账号并获取 API Token。
